Instructions for use:
	1.	Compile client.java and server.java
	2.	Run server, then run clients
	3.	We have included a humorous "Test.jpg" for testing the ./sendimage feature

Commands:
	quit		=	client leaves server
	./sendimage	= 	client prepares to send image to other connected clients
	./accep		= 	receiving client accepts and downloads image
	./reject	= 	client rejects image and continues chatting normally
	
Please enjoy the app!
